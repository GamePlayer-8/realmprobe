# Code of Conduct

This Code of Conduct is about to guide you through, how to contribute to the RealmProbe repository and explain the ethics regarding the communication within the repository's space.

# Tl;dr

We do not tolerate:

- Threats of hurting anyone
- Stalking
- Spam
- Phising of other's users informations or repository's secrets keystore
- Doxxing
- Nazism, facism and anything similar
- Homophobia, transphobia and anything similar
- Any other kinds of harassment

Please respect each other and be open & wisely minded on implementing new things / ideas.

## Reporting incidents

If you think someone is violating the Code of Conduct, you can report it by emailing [gameplayer2019pl@tutamail.com](mailto:gameplayer2019pl@tutamail.com). In the message please describe who you are and what did just happened, along with the link to the incident.

# Termination

Anyone who's violating the Code of Conduct, once declining to stop doing the harmful action(s), will be blocked from participating & communicating within the RealmProbe repository.
Any harmful action(s) will be reported as well to the upper management (git instance owners).
